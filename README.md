# Juego de Plataforma estilo Mario Bross – 3 Niveles (Pygame)

## Descripción del Proyecto
Este es un juego de plataformas desarrollado en Python usando Pygame.  
El jugador debe avanzar a través de **3 niveles**, recolectando monedas, evitando enemigos, saltando entre plataformas y llegando a la meta de cada nivel.  
El juego incluye físicas básicas, movimiento lateral, salto, cámara que sigue al jugador, vidas, puntajes, enemigos patrullando y mensajes de nivel completado o game over.

---

## Cómo se debe jugar

### Controles
- Moverse a la izquierda: `A` o `←`
- Moverse a la derecha: `D` o `→`
- Saltar: `W`, `↑` o `Espacio`
- Reiniciar el juego: `R`

---

### Objetivo
Completar los **3 niveles** llegando al bloque dorado marcado como `G`.

### Reglas
- Tienes 3 vidas.
- Perderás una vida si caes al vacío o un enemigo te golpea.
- Puedes eliminar enemigos saltando sobre ellos.
- Las monedas otorgan puntos.
- Completa los niveles para ganar.
